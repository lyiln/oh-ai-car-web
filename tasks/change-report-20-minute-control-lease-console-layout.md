# 20 分钟控制租约与控制台布局变更报告

## 来源与目标

执行 2026-07-14 已批准方案：控制权回归有时限租约，租期延长为 20 分钟并移除网关心跳；同时优化控制台地图、视频、方向控制和车牌识别工作台布局。

## 已实施变更

- `control_leases` 保持单车唯一活动记录，`expires_at` 恢复为必填；迁移会为临时无到期时间的活动记录设置 20 分钟过渡期，并删除临时心跳字段。
- 前端每 5 分钟续租，并用 `leaseRefresh` 将新令牌和到期时间同步给本机网关。
- 网关按租约到期时间设置本地定时器；到期或续租失败时先尝试发送 Stop，再关闭 TCP。页面关闭、显式断开、探测失败和连接失败也会尽力释放租约。
- 控制台顶部高德地图调整为 360px；中部左侧依次为视频预览和完整车牌识别工作台，右侧为等高的连接状态与控制区，消除左下方大块空白。
- 视频预览底部新增“车牌识别”入口，点击后滚动到原有工作台；识别内容没有并入视频卡片，也没有删除原工作台。
- 精简 YOLO 未启动及违规车辆操作提示，删除容易造成误解的“测试用”文字。

## 边界

本次不修复车牌识别推理流程。iframe 加载和自动化测试不证明真实视频流、识别准确率或真车控制协议已验证；真车验证仍以 `PROTOCOL_STATUS.md` 和相应实车记录为准。
