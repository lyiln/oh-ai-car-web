# Reference Boundary

This repository is a self-contained Web implementation. It can be cloned,
built, tested, and uploaded without a local copy of the OpenHarmony project.
The original OpenHarmony project is retained separately as historical source
evidence and is never built or modified from this workspace.

- Web specification authority: `specs/001-web-control-gateway/`

The source-analysis document is an evidence snapshot, not a file dependency.
Any protocol discrepancy found during hardware validation must be recorded
before changing the Web contract.
