# Source Evidence Provenance

The original OpenHarmony smart-car source is preserved outside this repository.
This Web repository contains no copied ArkTS application code and requires no
local path to it.

The included source analysis was created from reference commit
`6a9a7cb8839a6c16777eabf1f74e65d8c5867c1f`. It documents the TCP protocol,
video endpoint, and command evidence that informed Web v1. It is retained for
auditability when this repository is cloned independently.
